# Phase 0 - Project Setup

## Goal

The goal of Phase 0 is to create a clean starting point for the Exam Calendar project.

By the end of this phase, the repository should contain a runnable frontend application, a runnable backend application, basic documentation, and a simple Git workflow.

## Scope

Phase 0 includes:

* Creating the GitHub repository
* Creating the basic folder structure
* Adding initial documentation
* Setting up the React TypeScript frontend
* Setting up the Java Spring Boot backend


## Tools

Planned tools:

* VS Code for frontend development
* IntelliJ IDEA for backend development
* Git for version control
* GitHub for remote repository hosting
* Node.js for the frontend toolchain
* JDK 21 for the backend

## Definition of Done

Phase 0 is complete when:

* The GitHub repository exists
* The README explains the project idea
* The `docs/` folder contains basic planning notes
* The frontend application can start locally
* The backend application can start locally
